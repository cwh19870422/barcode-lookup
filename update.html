1上傳更新後的 Excel 檔案

可以一次選多個檔案。系統會依檔名判斷分類（例如「新版01沙龍美髮.xls」→ 沙龍美髮），並取代該分類原本的資料，其他分類保持不變。目前的資料會先從這個網站的 data.json 讀取，再套用你的更新。

拖曳檔案到這裡，或點擊選擇 Excel 檔案
支援 .xls / .xlsx，可多選

2下載新的 data.json

確認上面的更新記錄沒問題後，下載新的資料檔。

3上傳到 GitHub，覆蓋舊檔案

1. 打開你的 GitHub repo 頁面
2. 點進舊的 data.json 檔案，右上角點鉛筆圖示（Edit）或用「Add file → Upload files」重新上傳同名檔案
3. 選擇剛剛下載的新 data.json，上傳並蓋掉舊檔
4. 下方填寫 commit 訊息（例如「更新條碼資料 2026-07-23」），點 Commit changes
5. 等 30 秒到 1 分鐘，GitHub Pages 會自動重新部署，你的固定網址（index.html）就會顯示最新資料 — 不需要改任何設定，網址完全不變。
